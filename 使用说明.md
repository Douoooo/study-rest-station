# 使用说明

## 安装 Node.js

在使用此项目前，请先安装 Node.js：

1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本（推荐 v18 或更高版本）
3. 安装完成后，在终端输入 `node -v` 验证安装成功

## 启动项目

### 1. 安装依赖
```bash
cd "考研无痛久坐搭子"
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 在浏览器中打开
开发服务器启动后，终端会显示本地地址（通常是 `http://localhost:5173`），在浏览器中打开即可。

## 项目功能概览

### 1. 久坐无声拉伸库 🧘
- 12个拉伸动作，分为肩颈/腰背/手腕/腿部四大类
- 每个动作≤30秒，适合自习室使用
- 详细步骤、小贴士、示意图
- 支持下载动作卡片

### 2. 无痛久坐规划表 📊
- 4种学习模式：基础/冲刺/轻松/全天
- 可视化时间安排
- 支持下载打印

### 3. 自习坐姿优化指南 🪑
- 5大模块完整指南
- 快速自检清单
- 疼痛缓解方法

### 4. 久坐提神小技巧 ⚡
- 12种提神技巧（无咖啡因）
- 6种护眼动作
- 按时间段推荐
- 计时功能

### 5. 护眼轻动作 👁️
- ≤20秒快速护眼
- 无声音设计
- 6种不同动作

## 特色功能

✅ **自习室友好**：所有动作无声、小幅度
✅ **无需额外工具**：利用书本、靠垫等日常物品
✅ **可下载打印**：规划表和动作卡片可打印贴在书桌
✅ **响应式设计**：支持电脑、平板、手机访问
✅ **精美UI**：现代化设计，操作流畅

## 推荐使用流程

1. **开始学习前**：
   - 查看「坐姿指南」，调整好桌椅高度和坐姿
   - 选择适合的「学习规划表」并下载打印

2. **学习过程中**：
   - 每45分钟休息5分钟
   - 从「拉伸库」选择1-2个拉伸动作
   - 如果感觉困倦，使用「提神小技巧」

3. **遇到不适时**：
   - 腰部不适：查看「坐姿指南」的疼痛缓解方法
   - 眼睛疲劳：进行「护眼轻动作」
   - 精神不振：使用「提神小技巧」

## 注意事项

- 所有拉伸动作要缓慢进行，避免用力过猛
- 如果感到不适，立即停止
- 保持正确的坐姿比拉伸更重要
- 建议每30-45分钟休息一次

## 技术支持

如有问题，请检查：
1. Node.js 是否正确安装
2. 是否在项目目录下运行命令
3. 端口是否被占用
