# 考研/考公党「无痛」久坐搭子

专为20-26岁考研/考公党设计的久坐舒适化解决方案，所有功能围绕自习室/图书馆场景设计（无声、小幅度、无工具）。

**🎨 米黄色主体 + 淡粉色辅助 · 温柔舒适的学习体验**

## 解决的核心问题

打开应用即可看到我们解决的核心痛点：

- 😣 **久坐腰酸背痛** - 长时间学习导致的身体不适
- 😵 **颈椎眼睛酸痛** - 坐姿不当带来的健康问题  
- 😴 **学习注意力不集中** - 疲劳影响学习效率

## 核心功能

### 1. 久坐无声拉伸库 🧘
- **分类系统**：肩颈 / 腰背 / 手腕 / 腿部四大类别
- **动作特点**：每个动作≤30秒，小幅度、无声音
- **详细指南**：高清静态图 + 文字步骤 + 专业建议
- **场景适配**：完全贴合自习室/图书馆使用限制
- **下载支持**：支持下载动作卡片

### 2. 无痛久坐规划表 📊
- **时间模板**：基础学习模式 / 冲刺强化模式 / 轻松复习模式 / 全天规划模板
- **适配节奏**：针对考研/考公的学习节奏设计
- **灵活调整**：冲刺阶段拉伸时间可缩短至3分钟
- **下载打印**：支持图片版下载，可贴在书桌
- **可视化**：清晰的时间安排和循环次数

### 3. 自习坐姿优化指南 🪑
- **场景化指导**：利用书本/靠垫调整坐姿，无需额外工具
- **五大模块**：
  - 椅背调整（腰靠位置、椅背角度）
  - 桌面高度（桌面与手臂关系）
  - 身体姿态（脊柱、肩膀、头部）
  - 腿部位置（双脚、膝盖、大腿）
  - 自习室特殊技巧
- **快速检查**：每30分钟快速自检清单
- **疼痛缓解**：腰部/肩颈/手腕疼痛的快速缓解方法

### 4. 久坐提神小技巧 ⚡
- **无咖啡因方案**：适配自习室不能吃东西/喝饮料的场景
- **无声方法**：捏虎口 / 快速眨眼 / 深呼吸 / 太阳穴按摩等
- **时间推荐**：上午/下午/晚上的最佳提神技巧
- **计时功能**：每个技巧建议时长，可设置定时器

### 5. 护眼轻动作 👁️
- **≤20秒动作**：快速护眼，不浪费时间
- **无声音设计**：自习室友好
- **多样动作**：远近交替 / 眼球转动 / 热敷眼睛 / 画8字等
- **详细步骤**：每个动作都有清晰的执行步骤

## 推荐使用流程 🎯

### 首次使用（5步完整体验）
1. **打开应用首页** → 查看完整使用流程指南
2. **调整坐姿**（5分钟）→ 打开「坐姿指南」模块，按照5大模块调整桌椅和坐姿
3. **规划时间**（3分钟）→ 打开「规划表」模块，选择适合的学习模式并下载打印
4. **选择拉伸**（随时）→ 打开「拉伸库」，熟悉各部位拉伸动作
5. **开始学习** → 按照规划表开始学习，休息时使用拉伸和提神技巧

### 日常使用循环
- 学习45分钟 → 选择1-2个拉伸动作（每个≤30秒）
- 感到困倦 → 使用提神技巧（10-30秒）
- 身体不适 → 查看坐姿指南的疼痛缓解方法
- 眼睛疲劳 → 进行护眼轻动作（≤20秒）

## 功能模块说明

### 拉伸库模块
- 12个精心设计的拉伸动作
- 每个动作包含：名称、时长、步骤、小贴士、示意图
- 支持按身体部位分类浏览
- 可展开/收起动作详情

### 规划表模块
- 4种不同学习模式
- 可视化时间安排
- 支持展开/收起完整日程
- 可下载打印

### 坐姿指南模块
- 5大模块完整指南
- 快速自检清单
- 疼痛缓解方法
- 常见错误警示

### 提神技巧模块
- 12种提神技巧
- 6种护眼动作
- 按时间段推荐
- 模态框显示详情和计时功能
