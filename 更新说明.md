# 更新说明 - v2.0

## 更新内容

### 1. 🎨 全新配色 - 清新绿色系
- 将原有紫色系配色改为清新的绿色系
- 主色调：#66bb6a → #43a047 渐变
- 所有组件统一使用绿色主题，视觉更舒适
- 符合学习场景的舒缓氛围

### 2. 🏠 新增首页功能
#### 使用流程指南
- 4步推荐流程，清晰展示使用顺序
- 箭头连接，带有弹跳动画
- 每步显示所需时间
- 点击步骤可跳转到对应功能
- 底部提供完整使用建议

#### 功能模块预览
- 4个模块卡片，展示核心功能
- 大图标设计，视觉效果佳
- 显示3个核心特性标签
- hover时显示"查看"按钮
- 点击可直接进入对应模块

#### 快速开始按钮
- 大号主按钮，引导用户按推荐顺序开始
- 渐变绿色背景，醒目易用
- 点击后自动跳转到坐姿指南模块

### 3. 📱 导航优化
- 新增"首页"导航项，默认显示首页
- 5个导航项：首页、拉伸库、规划表、坐姿指南、提神技巧
- 点击导航项平滑滚动到顶部

### 4. ✨ 用户体验提升
#### 更清晰的使用动线
- 首页展示完整使用流程，无需猜测
- 推荐顺序：调整坐姿 → 规划时间 → 选择拉伸 → 使用提神技巧
- 每个模块都有明确的功能说明和时间预估

#### 更直观的功能展示
- 模块卡片展示核心功能，点击即用
- 标签化展示功能特点（如"4种模式"、"12个动作"）
- hover效果增强交互反馈

#### 更好的引导体验
- 首页提供完整的新手引导
- 流程指南可快速跳转到任意功能
- 快速开始按钮一键进入推荐流程

## 使用建议

### 新用户
1. 打开应用，首页查看完整使用流程
2. 点击"按推荐顺序开始使用"或按步骤逐个体验
3. 调整好坐姿后，选择合适的规划表
4. 开始学习时参考规划表和拉伸库

### 老用户
1. 导航到"首页"查看更新内容
2. 了解推荐的使用流程
3. 尝试按照新流程使用，效果更佳
4. 享受全新的绿色视觉体验
